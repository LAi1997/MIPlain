Introduction

Do you want to become an expert of the Island game?

To do so, you will pass a trial which includes 4 parts.
You will first get familiar with the rules of the game.
Then you will answer some questions about single moves.
Next you will become an apprentice of the Great Wizard.
Finally, you will apply what you have learnt by answering some more questions about single moves.

Rules of the game


First, here are some basics of the game. The Island game is a two-player game.
The two players, Green and Orange, take turns conquering territories.
Any untaken territory can be conquered until the game terminates.
You will play Green.

Conquering a territory provides one or several resources.
Below is the initial world. Select one territory to get some resources.




Each resource belongs to one of the following category:
Island 1, Island 2, Island 3, Forest, Castle, Cornfield, River, Animal.

You can press the button at the corner of each board to view the table for resources captured.
Pressing "P1 resources" will show the resources captured by Green so far,
then pressing "P2 resources" will show the resources captured by Orange so far.
Pressing "Board" will get you back to the current board.
