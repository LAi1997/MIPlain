Introduction

Do you want to become an expert of the Island game?

To do so, you will pass a trial which includes 4 parts.
You will first get familiar with the rules of the game.
Then you will answer some questions about single moves.
Next you will become an apprentice of the Great Wizard who is a master of the island game.
Finally, you will apply what you have learnt by answering some more questions about single moves.

Rules of the game

First, here are some basics of the game.
The game is played on the board below. There are three islands:
Island 1, Island 2, Island 3.
Each island consists of three territories.
Each territory can contain one, two or three different resources. Resources are:
Forest, Castle, Cornfield, River, Fish.


Any untaken territory can be conquered until the game terminates.
For a given constellation of islands, you can capture some resources by clicking on a territory containing them.
Select one territory (white cell) on the board above to capture some resources.
